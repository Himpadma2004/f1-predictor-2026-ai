# F1 Predictor 2026 (AI & ML Ecosystem)

### Core Objective
Predict race outcomes for the 2026 regulation era by combining live telemetry, 
historical driver skill, and real-time news analysis.

### Tech Stack
- **Frontend:** React + Tailwind CSS (The Dashboard)
- **Backend:** Python + FastAPI (The Logic)
- **AI/ML:** Scikit-Learn (Historical), PyTorch (Live), Gemini API (Strategy/News)
- **Data:** OpenF1 API (Live), Jolpica (History), NewsAPI (Context)

### Deployment
- **Frontend/Backend:** Render / Railway (Dockerized)
- **Version Control:** GitHub
